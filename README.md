# Realtime Layout

Create TSNE or UMAP layouts with JavaScript in realtime

# Usage

To use this boilerplate, you can run in a terminal:

```bash
git clone https://github.com/yaledhlab/realtime-layout
yarn install
```

Then one can start the server:

```bash
yarn start
```
